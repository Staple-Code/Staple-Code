Controller constructor creates an instance of Staple_View and saves it in the $view property. It then calls the overridable method _start() for additional boot time procedures.
Staple_Controller::index()
The default action on any controller must be defined.
Staple_Controller::_auth()
Returns a boolean true or false whether the method requires authentication before being dispatched from the front controller.
Staple_Controller::_authLevel()
Returns the access level required for this method.
Staple_Controller::_openAll()
This function opens the entire controller to be accessed without authentication.
Staple_Controller::_openMethod()
Sent a string it allows one method to be accessed without authentication. When sent an array, it allows all the values method names without authentication.
Staple_Controller::_redirect()
This function accepts a routing string to redirect the application internally. A redirect of this sort clears the output buffer and redraws the header, proceeding as if the redirected controller/action was called directly.
Staple_Controller::_requiredActionAccessLevel()
Specifies the required access level for the specified action. This should be called in the Controller startup.
Staple_Controller::_requiredControllerAccessLevel()
Replaces the default permission level with the specified permission level. All method specific access levels will be overwritten. This should be called in controller startup.
Staple_Controller::_start()
Overridable and empty method allows for additional boot time procedures.
