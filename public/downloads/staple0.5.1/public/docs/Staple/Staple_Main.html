Class: Staple_Main

Source Location: /Main.class.php

Class Overview




The main class for the STAPLE Framwork, Staple_Main creates the application and  coordinates all the other modules. Using this object will require the entire  framework code to be available. Many other modules can stand apart with varying  levels of autonomy.
