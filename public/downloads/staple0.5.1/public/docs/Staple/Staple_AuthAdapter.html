Class: Staple_AuthAdapter

Source Location: /AuthAdapter.class.php

Interface Overview




The Staple_AuthAdapter interface creates a preset list of functions that must be implemented  to allow authentication to pass through Staple_Auth. Adapters sent to Staple_Auth must  implement this class or an error will occur.
