Class Details

[line 31]
The Staple_DB class extends the MySQLi class to give added functionality. Database configuration file requires these fields:

host - the mysql hostname of the database server to connect to. username - MySQL username to use to connect to the database. password - MySQL password to use to connect to the database. db - the database to bind to.
